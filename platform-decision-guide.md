# Zequence Digital UK Platform Decision Guide

Use this guide when deciding whether a client project should be built as a static website, a Shopify store, or a WordPress / WooCommerce site.

## 1. Static Website

Best for:

- Service businesses
- Brochure websites
- Personal brands
- Businesses that mainly need credibility, enquiries, and a clean online presence

Use this when:

- the client does not need to manage products
- the client does not need to change content often
- the site is mainly informational
- updates can be handled by you

Pros:

- fast to build
- lightweight
- premium design flexibility
- low maintenance

Cons:

- no easy admin for the client by default
- content updates usually need you unless a CMS is added

## 2. Shopify

Best for:

- product-based businesses
- fashion brands
- jewellery businesses
- sellers moving from Instagram, eBay, or marketplaces into their own branded store

Use this when:

- the client wants to sell products online
- the client wants admin access for products, prices, and store management
- the client needs a reliable checkout and store backend
- the client wants a platform that is easier to manage over time

Pros:

- strong eCommerce admin
- easy product and price updates
- reliable checkout
- easier for clients to use
- good for long-term store growth

Cons:

- monthly Shopify subscription is separate
- paid apps or premium themes can add cost

## 3. WordPress / WooCommerce

Best for:

- clients who want a non-Shopify website with admin access
- businesses that need editable pages and potentially products
- projects where WordPress flexibility is more suitable

Use this when:

- the client wants to edit the website themselves
- the project is non-Shopify but still needs admin access
- a brochure site may later grow into something more dynamic

Pros:

- client can log in and manage content
- flexible for different site types
- can support products through WooCommerce

Cons:

- more maintenance than static websites
- more technical upkeep than Shopify
- more things can break if not managed properly

## Quick Decision Rule

- Use `Static Website` for service sites and brochure websites
- Use `Shopify` for product-selling businesses that need a proper store
- Use `WordPress / WooCommerce` when a non-Shopify client still needs admin access

## Recommended Client Language

### Static Website

"A static website is best when you need a clean, professional online presence and do not need to manage products or regular content updates yourself."

### Shopify

"Shopify is best when you want to sell products online and manage things like products, prices and orders through an easy admin dashboard."

### WordPress / WooCommerce

"WordPress is best when you want a non-Shopify website but still need admin access to update pages, content or products yourself."
