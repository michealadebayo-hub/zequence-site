## 6. Platform Recommendation Rule

- Use `Static Website` for brochure/service websites where you handle updates
- Use `Shopify` for product-selling businesses that need a real store and admin access
- Use `WordPress / WooCommerce` for non-Shopify clients who still need admin access

Reference:

- [platform-decision-guide.md](/home/mikeoye/zequence-site/platform-decision-guide.md)

## 7. Simple Conversation Flow

1. Lead asks about pricing
2. Send pricing message or rate card
3. Ask discovery questions
4. Decide platform and scope
5. Send proposal
6. Client approves
7. Send onboarding questionnaire
8. Begin project
